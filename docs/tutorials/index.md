# Tutorials

Learning-oriented, start-to-finish walkthroughs. If you are new here, do these in order. Each one takes you from nothing to a working result on your own machine, with every command spelled out.

- **[Your first private AI platform](../quickstart.md)**: bring up the local `kind` lab, deploy the platform, and get a real private chat completion back in 15 to 30 minutes.

Once you have run the tutorial, the **[How-to guides](../getting-started.md)** show how to do specific tasks (deploy to a customer cluster, run evals, generate evidence), the **[Reference](../production-readiness.md)** documents the contracts and controls, and the **[Explanation](../decision-guide.md)** pages cover the why.
